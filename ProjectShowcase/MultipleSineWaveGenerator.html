Multiple Sine Wave Generator

Verilog, DE1-SoC, LT24 Display,
FPGA, CORDIC Algorithm

March 2023 - May 2023 (7 weeks)

This group project involved working in a team of three to create a Multiple Sine Wave Generator Instantiated using FPGA Hardware. Edward's responsibilities focused on the Sine Wave Generation. He was provided with an Add Shift Module and used this to create a series of Sine Wave Generators generating points using the CORDIC Algorithm. The project involved Self Verifying Test Benches, Parameterisation of Modules and Pipelining.
