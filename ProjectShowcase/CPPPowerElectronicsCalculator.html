Power Electronics Command Line Based Calculator

Name:

Language/Tools Used:

Timeframe

SwitchingDevicesCalculator

C++ Command Line

December 2020 (1 Month)

A group project where I developed a Calculator that helps you answers questions related to the topic of Power Electronics. The aim of the project was to come up with a technical engineering tool in groups of three and develop a command line based program. My group settled on a Power Electroncis calculator and we each took a part of that topic and designed a tool to aid with that. My calculator was based around: Switching Devices. The program was written in C++, included file reading functionality, Object Orientated Programming, Multiple layer menus and search functionality
